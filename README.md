# Angular CRUD Application (Standalone Architecture)

## Overview

This project is a simple User Management CRUD application built using Angular (Standalone Components).

It demonstrates:

- Reactive Forms
- Angular Material UI
- Feature-based architecture
- RxJS Observables
- Clean folder structure

## Tech Stack

- Angular 17+
- TypeScript
- Angular Material
- RxJS

## Features

- Create User
- Edit User
- Delete User
- View User Details

## Architecture

Feature-based folder structure with standalone components.

## How to Run

npm install
ng serve
